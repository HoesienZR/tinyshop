# TinyShop

TinyShop is a learning-focused backend project in Python. The goal is to build a scalable and maintainable e-commerce engine while practicing software architecture and clean code principles.

## Current Focus
I am currently working on the core logic and architecture. My main goal right now is to ensure the code is modular and easy to test before integrating the persistence layer.
