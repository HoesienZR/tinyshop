# Contributing to TinyShop

First of all, thank you for taking the time to review this project! 

As this is a learning-focused repository, your feedback is incredibly valuable to me. Here is how you can help:

## Code Reviews & Suggestions
* If you spot an anti-pattern, open an issue explaining the anti-pattern, why it is problematic, and how to refactor it (pointing to the specific lines of code).
* Feel free to submit a Pull Request with the proposed changes. Please include comments explaining *why* the new approach is better.

## Technical Discussions
If you want to discuss architectural decisions (e.g., choosing relational vs non-relational, structuring folders, async operations), please open an issue with the tag `discussion`.
